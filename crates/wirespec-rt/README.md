# wirespec-rt

Runtime support for Rust code generated by [wirespec](https://github.com/wirespec-lang/wirespec).

Provides `Cursor` (zero-copy reader), `Writer`, `Error`, and checksum functions (internet, CRC32, CRC32C, fletcher16) used by generated parse/serialize code.

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
wirespec-rt = "0.3"
```

This crate is required when using wirespec-generated Rust code.

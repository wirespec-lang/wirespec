# wirespec-backend-c

C code generation backend for [wirespec](https://github.com/wirespec-lang/wirespec).

Generates `.h` and `.c` files with parse, serialize, and serialized_len functions from wirespec protocol definitions.

This is an internal crate used by the wirespec compiler. For the user-facing tool, see [`wirespec`](https://crates.io/crates/wirespec).

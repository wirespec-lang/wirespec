# wirespec-backend-tlaplus

TLA+ code generation backend for [wirespec](https://github.com/wirespec-lang/wirespec) state machines.

Generates `.tla` specs and `.cfg` configs for formal verification with the built-in model checker or external TLC.

This is an internal crate used by the wirespec compiler. For the user-facing tool, see [`wirespec`](https://crates.io/crates/wirespec).
